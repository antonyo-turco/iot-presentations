\begin{column}{0.55\textwidth}
\begin{block}{What Happens on Alarm}
After a valid knock is classified with confidence above threshold, the Central Hub orchestrates a multi-stage response.
\end{block}
\vspace{0.2cm}
\begin{enumerate}
\item \textbf{Siren}: active buzzer on the hub fires immediately
\item \textbf{Floodlights}: optional relay-controlled lights
\item \textbf{Deadbolt}: optional electronic lock engagement
\item \textbf{Neighbor wake-up}: ESP-NOW broadcast wakes adjacent nodes for richer contextual sensing
\item \textbf{Cloud alert}: MQTT publish $\rightarrow$ dashboard notification $\rightarrow$ push to user phone
\end{enumerate}
\end{column}
