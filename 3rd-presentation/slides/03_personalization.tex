  \begin{itemize}\setlength{\itemsep}{4pt}\setlength{\parskip}{0pt}
  \item \textbf{Alarm rate:} up to 12 events/hour
  \item \textbf{Active time per alarm:} ${\sim}3\,\text{s}$
        (2.56\,s ADC @ 200\,Hz, 512 samples + $<200$\,ms FFT + TX)
  \item $12 \times 3\,\text{s} = 36\,\text{s/hr}$
        $\Rightarrow$ \textbf{duty cycle: 1\%}
\end{itemize}

\vspace{0.2cm}

INT1 fires on threshold crossing; MCU wakes via GPIO.
